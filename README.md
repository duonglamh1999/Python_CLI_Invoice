# Python_CLI_Invoice
It is a tool to track your daily hours, tasks also pairs with PDF generating ability.
Functionalities: 
[LOG]: Log/edit your hours and tasks
[READ]: Read current CSV files
[POPULATE]: Populate a PDF invoice
